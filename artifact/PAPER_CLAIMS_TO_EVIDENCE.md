# Paper Claims to Evidence Map

This table maps major paper-facing numerical claims to concrete artifact evidence so
that every headline value is directly inspectable.

## RQ1 count authority and row-level crosscheck

The 457-row registry records raw Stage-3 extraction records before claim-level
correction. The paper-facing final accepted-claim count is 455, supported by
`corrected_headline_counts.csv` and `source_grounded_claim_adjudication.csv`.
The derived `field_study/claims/rq1_claim_decisions_457.csv` applies the base
relation flags and 16-row source-grounded crosscheck to all candidate rows so that
reviewers can audit the decisions row by row. It crosschecks **455** accepted
comparative claims, **175** claim-card-specifiable, **145** scalar-directional,
**93** planning-feasible, **79** scalar-oriented planning records, **53** broader
auditable designs (8 exact + 2 source-materialized + 43 extensions), **8**
proxy-free exact cards, and **30** non-scalar relation-typed specifiable claims
(**26** multi-objective + **2** categorical/equality + **2** crossover/regional);
paradigm split **142** compilation / **313** variational.
`field_study/corpus/final_rq1_paper_registry_119.csv` records all 119 included
papers, 118 of which contribute accepted claims. The 76-row materialization-tier
census is retained as a historical pre-adjudication input.

Status values:

- `VERIFIED`: source file exists and the sanity script verifies the value.
- `PARTIAL`: source evidence exists, but the claim depends on a derived summary or reviewer inspection rather than a single raw table.
- `STATIC CLAIM`: conceptual/static table, not data-derived.

Checker commands are read-only by default. Output paths under `artifact/outputs/reports/`
are optional and are produced only with the checker's `--write-report` flag.
